Viewer API Dokumentáció

1. Prezentáció indítása

URL: POST /:id/start

Leírás: Elindít egy prezentációt a megadott azonosítóval.

Paraméterek:

  • id (URL paraméter) - A prezentáció azonosítója.

Válasz:

  • Sikeres válasz: 200 OK + a prezentáció adatai.
  • Hiba esetén: 500 Internal Server Error + hibaüzenet.

2. Következő dia

URL: POST /:id/next

Leírás: Átvált a következő diára a prezentációban.

Paraméterek:

  • id (URL paraméter) - A prezentáció azonosítója.

Válasz:

  • Sikeres válasz: 200 OK + az új dia adatai és tartalma.
  • Hiba esetén: 500 Internal Server Error + hibaüzenet.

3. Előző dia

URL: POST /:id/previous

Leírás: Átvált az előző diára a prezentációban.

Paraméterek:

  • id (URL paraméter) - A prezentáció azonosítója.

Válasz:

  • Sikeres válasz: 200 OK + az új dia adatai.
  • Hiba esetén: 500 Internal Server Error + hibaüzenet.

4. Ugrás egy adott diára

URL: POST /:id/goto

Leírás: Átvált a megadott sorszámú diára.

Paraméterek:

  • id (URL paraméter) - A prezentáció azonosítója.
  • gotoSlide (Body paraméter) - A dia sorszáma, amire ugrani szeretnénk.

Válasz:

  • Sikeres válasz: 200 OK + az új dia adatai.
  • Hiba esetén: 500 Internal Server Error + hibaüzenet.

5. Prezentáció leállítása

URL: POST /:id/stop

Leírás: Leállítja a futó prezentációt.

Paraméterek:

  • id (URL paraméter) - A prezentáció azonosítója.

Válasz:

  • Sikeres válasz: 200 OK + a leállítás megerősítése.
  • Hiba esetén: 500 Internal Server Error + hibaüzenet.
